# 渲染效果

#### 1.前言

- 渲染效果本身可以说是综合了物理，摄影，心理，统计，近似各方面学术的成果
- 在研究渲染效果时可以补全物理现象为基础，然后人为增强视觉效果
- 游戏中的渲染效果由于时耗能耗的要求，往往需要选择各种近似方案对各种性能与效果间的平衡

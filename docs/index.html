简介

个人文档集

1.前言

  • 文档主要记录个人学习知识点整理整合，目的在整理梳理学识的积累

  • 文档在内容记录分两种形式进行：

    graph LR A(基于理论)-->B(满足需求)
    graph LR A(基于需求)-->B(常用方案) B-->C(涉及理论) B-->D(实践优缺)

  • 结构会使用以下固格式记录:

    graph LR A("前言(偏向限制)") -->B("个人理解(对于理论/需求的本质理解)") B-->C("其他(记录形式)") C-->D(总结)
